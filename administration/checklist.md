# IT Administration Checklist

---

## Weekly Tasks 


1. **Check sign-in logs** — Entra ID → Sign-in logs → Look for failed logins or unknown locations
2. **Test email** — Send a test email to info@ and confirm it arrives
3. **Check website** — Visit guestturnoversolutions.co.uk and make sure it loads
4. **Check Secure Score** — M365 Admin Center → Security → Note the score

---

## Monthly Tasks


1. **Test break-glass account** — Sign in with BreakGlass-Admin, then sign out
2. **Check DNS records** — Namecheap → Advanced DNS → Make sure nothing changed
3. **Run MXToolbox test** — mxtoolbox.com → Confirm all tests pass
4. **Review Message Center** — Admin Center → Health → Message Center
5. **Update maintenance log** — Add monthly summary to log

---

## Quarterly Tasks 


1. **Change break-glass password** — Update and store securely
2. **Check domain expiry** — Namecheap → Make sure domain won't expire
3. **Full MXToolbox test** — Confirm 29/29 still passing
4. **Review website content** — Check if prices or info need updating

---

## If Something Goes Wrong

| Problem | What To Do |
|---------|------------|
| Suspicious sign-in | Block user, change password, check what was accessed |
| Email not delivering | Check DNS records, run MXToolbox test |
| Website down | Check GitHub repo settings, check DNS |
| Forgot admin password | Use break-glass account |
